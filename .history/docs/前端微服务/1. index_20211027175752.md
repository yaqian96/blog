# 1. 概念

```js
将微服务用于浏览器端,
即将 Web 应用由单一的单体应用转变为多个小型前端应用聚合为一的应用。各个前端应用还可以独立运行、独立开发、独立部署。
```

# 2. 特点

```js
独立运行、独立开发、独立部署
```
