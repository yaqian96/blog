# 9. 高维护性的 js

```js
1. 养成良好的编码习惯，提高代码的可维护性
   1. 避免定义全局变量或函数
   2. 使用简化的编码方式
   3. 使用比较运算符===而不是==
   4. 避免使用with语句
   5. 避免使用eval
   6. 不要编写检测浏览器的代码
2. 使用更严格的编码格式
   1. 不要在全局中启用严格模式
   2. 在已有代码中谨慎启用严格模式
3. 事件处理和业务逻辑分离
4. 配置数据和代码逻辑分离
5. 逻辑与结构样式分离
6. js模板的使用
   mustache Underscore Handlebars
7. 在js开发中应用MVC模式
8. js模块化开发
9. 合理使用AJAX技术
```

# 10. 高性能的 js

```js
1. 加快js文件的加载速度
   async defer
2. 养成良好的编码习惯，提高代码运行速度
  1. 少用for-in循环
  2. 谨慎使用eval
  3. 正确使用数组
  4. 正确的内存回收
     1. 确保解除不需要的监听事件
     2. 不要在闭包中返回外部不需要的对象
     3. 尽量不适用全局变量
3. 使用高性能的变量或者属性值读取方式
4. 高效的DOM操作
   1. 合并多次的DOM操作为单次的DOM操作
   2. 把DOM元素离线或者隐藏后修改
     大批量修改DOM元素的情况
     1. 使用文档片段
     2. 通过设置DOM元素的display样式为none来隐藏元素
     3. 克隆DOM元素到内存中
   3. 设置具有动画效果的DOM元素的position属性为fixed或absolute
   4. 谨慎取得DOM元素的布局信息
   5. 使用事件托管方式绑定事件
5. 使用辅助工具优化js代码性能
```

# 11. 高安全性的 js

```js
1. 常见的web前端攻击方式
   1. XSS
   2. CSRF
   3. 界面操作劫持
2. 不要轻易信任何外部传入的数据
3. 其他前端安全防范实践
   1. 更安全使用cookie
   2. 防止网页被其他网站内嵌为iframe
```

# 12. 移动 Web 前端开发最佳实践

```js
1. 移动Web前端开发
   1. 如何让桌面Web页面兼容移动设备
      1. 流式布局
      2. 媒体查询技术
      3. 使用合适的图片显示兼容方案
      4. 设置viewport
      5. 增加链接和按钮的可操作区域
      6. 使用响应式设计框架
      7. 使用工具检查网页的移动设备兼容性
2. 移动Web前端开发相关技术最佳实践
   1. HTML相关最佳实践
     1. head添加必要的meta或link信息
     2. 利用移动设备的便利性
     3. 不使用iframe,谨慎使用table标签
   2. CSS相关最佳实践
     1. 尽量少使用图片，并使用内联图片代替小图片
     2. 不要使用：hover伪类设置悬停结果
     3. 设置合理的字体大小和行高
     4. 在不需要选中文字的区域禁用文字选中功能
   3. JS相关最佳实践
     1. 使用设备的原生事件
     2. 谨慎使用标准对话框，避免弹出窗口
     3. 谨慎使用定时器
```
