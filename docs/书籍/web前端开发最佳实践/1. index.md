# 2. 高效 web 开发

```js
1. 前端代码的组织结构和文件的命名
  1.1 按照不同模块进行划分
  1.2 代码文件的命名,见名知义
2. 前端代码重构
  2.1 删除无用代码
  2.2 前端代码规范化
  2.3. 整理基础类库
  2.4 前端代码模块化
  2.5 提高页面加载性能
3. 合理使用前端框架
4. 多浏览器的测试：多测试、早测试
5. 前端代码的开发与调试
   1. 代码的合并和压缩
      web服务器开启GZIP压缩
      js压缩：Uglify.js
      css压缩：合并样式、css Compressor
      HTML压缩
      图片压缩
6. 前端代码基本命名规范和格式规范
   1. HTML
   id用_连接、class用-链接、全用小写、缩进4个
   2. js
   变量用驼峰式
   常量用大写字母拼接
   分号的正确使用
```
