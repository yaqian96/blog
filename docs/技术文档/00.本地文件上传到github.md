# 上传步骤
1. 在github上创建一个仓库
2. 将仓库的链接复制
3. 在本地项目所在的地方，打开git bash,执行git clone 仓库地址
4. 本地会生成一个文件夹
5. 将本地项目的东西复制到文件夹内
6. git pull origin master
7. git add .
8. git commit -m '提交日志'
9. git remote add origin https://github.com/yaqian96/blog.git(仓库的地址)、
10. git push -u origin master
   *查看分支名(master/main)，git push -u origin 分支名
