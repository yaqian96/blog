# 隐式转换的场景
```js
1. 一元操作符 ++ --  +  -
   +false   //：布尔值 false和 true 转换为 0 和 1，字符串根据特殊规则进行解析，对象会调用它们的 valueOf()和/或 toString()方法以得到可以转换的值
   s=-s    //将其变换成对应的负值，遵守一元的操作
   ++ -- //将其进行相对应的数值转换进行计算
2. '+'
  1. 如果两个都是数值，则进行加法运算
  2. 如果其中有一个是字符串，则将另一个调用toString()转换成字符串拼接
3. '-' * /
  1. 如果两个都是数值，则进行减法运算
  2. 如果其中有一个是字符串，则将另一个调用Number()转换成数字
4. !
!运算符：将其他数据转换使用Boolean()转换成布尔类型，0 -0 NaN undefined null 空字符串 false document.all(),对应的都是false
5. 关系运算符 > < <= >=
  1. 两边有一个是字符串，则转换成number类型 console.log(2 > '10')
  2. 两边都是字符串时按照字符串对应的unicode编码 console.log('2' > '10') console.log('abc'>'a') //多个字符从左往右进行比较
  3. 两边都是数值
  4. 任一操作数为数值，则将另一个操作数转换为数值，执行数值比较。
  5. 则调用其 valueOf()方法，取得结果后再根据前面的规则执行比较。如果没有 valueOf()操作符，则调用 toString()方法，取得结果后再根据前面的规则执行比较。
  6. 如果有任一操作数是布尔值，则将其转换为数值再执行比较
     console.log(undefined==undefined) // true
     console.log(undefined==null) // true
     console.log(null==null) // true
     console.log(NaN==NaN)  //false
6. 逻辑非隐式与关系运算符隐式转换
//空数组的toString()方法会得到空字符串，而空对象的toString()方法会得到字符串`[object Object]`
console.log([]==0)  //true
console.log([]==0)  //true
console.log([]==![])  //true
console.log([]==[])  //false
console.log({}==!{}) //false
console.log({}=={}) //false

顺口溜：==
对象->字符串->数值 布尔值->数值
null==undefined
7. 判断语句 while if
   自动调用Boolean()
```
# 强制转换的场景
转换为数值类型：Number(mix)、parseInt(string,radix)、parseFloat(string)
转换为字符串类型：toString(radix)、String(mix)
转换为布尔类型：Boolean(mix)
